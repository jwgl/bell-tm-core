# 教学管理系统-核心模块

## 模块说明
- tm-uaa 用户帐户与权限
- tm-eureka 服务注册与发现
- tm-zuul 反向代理服务
- tm-report 报表服务
- tm-core-web 基础WEB服务
- tm-core-api 基础API服务

## 运行顺序
1. tm-uaa
2. tm-eureka
3. tm-zuul
4. 可选模块
    - tm-report
    - tm-core-web
    - tm-core-api
